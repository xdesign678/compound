### CSS 暗色模式顺序 Bug（影响 T01/T03/T04/T06/T07/T10）

- **问题**: `.dark {}` CSS 块定义在 `:root {}` 之前，两者同等优先级（0,1,0），后定义的 `:root` 覆盖了 `.dark` 的 CSS 变量，导致暗色切换失效
- **影响**: 所有 D1 评分标记为 SKIP
- **建议**: 将技能文档模板中 `.dark {}` 块移到 `:root {}` 块之后，或使用 `html.dark {}` 并确保优先级正确
- **记录到**: skill-issues.md
